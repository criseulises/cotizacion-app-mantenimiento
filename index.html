Introducción

Este análisis estima el tiempo necesario para desarrollar una aplicación móvil (iOS y Android) y una API en Java/Spring Boot que sirva como backend para manejar las funcionalidades descritas: inicio de sesión, verificación de identidad (cédula, Face ID), perfiles de usuarios y técnicos, sistema de cobro y pago, agendar citas, categorías de servicios (limpieza, mantenimiento, averías, mudanzas), y geolocalización en tiempo real de técnicos.

Consideraciones

  • Complejidad Alta: La aplicación incluye integración de mapas, pasarela de pagos, verificación biométrica y flujos tipo Uber.
  • Requerimientos de Personalización: App intuitiva, moderna y segura con backend eficiente y escalable.
  • Integración: Servicios externos como Stripe y Google Maps para sincronización en tiempo real.
  • Plazos Realistas: 16 semanas para la entrega del MVP (Octubre 2025 – Febrero 2026).
